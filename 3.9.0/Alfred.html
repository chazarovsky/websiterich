Customizable Analytics & Managed datalab

A través de Alfred de RICH IT

Acompañamos a nuestros clientes en una jornada de descubrimiento
y explotación del valor de la información

Alfred de RICH IT es un servicio administrado que permite contar con un equipo completo de ciencia de datos o de Ingeniería de datos. Cada célula analítica es capaz de atender un proyecto a la vez entregando valor continuamente.

Cada célula está compuesta de un científico de datos que se apoya de cualquier recurso especializado que se necesite según la etapa del ciclo analítico o componente de arquitectura. Por ejemplo: Ingeniero de datos, arquitecto de datos, desarrollador de software, experto en visualización, etcétera.

Data Engineering

  • Migración SAS, MATLab, SPSS, STATA a R, Spark o Hadoop
  • Migración de Datawarehouse, MPP y herramientas OLAP a Hadoop
  • Integración de datos, procesos de ETL, ELT, Datamart, Perfilado, calidad y limpieza de datos
  • Migraciones a SQL Server 2016.
  • Arquitectura, diseño e implementación de arquitecturas Kappa y Lambda para análisis en tiempo real y procesos por lotes.
  • JS para Front End y visualización de datos con D3 o visualizadores como PoweBI, QlickView y Tableau
