Elasticsearch

Elastic Stack contiene un motor de búsqueda empresarial que permite indexar y analizar grandes cantidades de datos en tiempo real. Permite la ingesta de datos estructurados y no estructurados y ofrece baja latencia, escalabilidad, flexibilidad y confiabilidad. Comprende los siguientes componentes:

Kibana: Te permite visualizar tus datos de Elasticsearch y navegar por el Elastic Stack.

Elasticsearch: Es un motor de búsqueda y análisis RESTful distribuido, capaz de resolver un número creciente de casos de uso. Como la base del Elastic Stack, almacena de forma centralizada tus datos, para que puedas descubrir lo esperado y descubrir lo inesperado.

Logstash: Es un canal de procesamiento de datos del lado del servidor de código abierto que ingiere datos de una multitud de fuentes simultáneamente, las transforma y luego las envía a su "stash" favorito.

Beats: Es la plataforma para transportadores de datos de un solo propósito. Se instalan como agentes ligeros y envían datos de cientos o miles de máquinas a Logstash o Elasticsearch.
